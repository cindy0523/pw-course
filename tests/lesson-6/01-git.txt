a. git clone <url>
Tải 1 repo về máy, nó sẽ clone toàn bộ các folder, chứa đẩy đủ code và lịch sử commit

b. git branch <branch_name>
Câu lệnh tạo 1 branch mới nhưng không chuyển sang branch đó ngay

c. git checkout <branch_name>
Câu lệnh chuyển sang branch mong muốn

d. git checkout -b <branch_name>
Câu lệnh tạo branch và chuyển sang branch đó ngay sau khi tạo

e. git pull origin master
Lấy code mới nhất từ nhánh master về + merge vào nhánh hiện tại
