Câu lệnh git commit --amend:
dùng để chỉnh sửa commit message gần nhất 

Câu lệnh git reset HEAD~3:
dùng để xóa 3 commit gần nhất, nhưng ko xóa file, các file đó sẽ nằm trong vùng working directory (đã add) và sẵn sàng commit lại

Liệt kê các branch, các commit và các file theo từng vùng ở các branch: main, feature A, feature B
sau khi thao tác các lệnh sau:
a. Tạo thư mục lesson-8
nằm trong local
---
b. Tạo file: file1, file2, file3
file1, file2, file3 nằm trong local, bên trong folder lesson-8
---
c. Chạy lệnh: git init
file1, file2, file3:
- vùng: working dir
- branch: main
---
d. Chạy lệnh: git add file1
file1:
- vùng: staging
- branch: main
file2, file3:
- vùng: working dir
- branch: main
---
e. Chạy lệnh: git commit -m”init project”
file1: 
- vùng: repository
- commit: ”init project”
- branch: main
file2, file3:
- vùng: working dir
- branch: main
---

- vùng: repository
- commit: ”init project”

- branch: feat/feature-a
